- ベイズ最適化の紹介の最後に獲得関数の話をする
  - 獲得関数とは，最大化をするために，次の観測点を決めるための指標になるものである
- 探索によりすぎるパターンと活用によりすぎるパターン(従来?平均だけを見るだけのものと分散だけを見るだけのものが考えられることを話す)
